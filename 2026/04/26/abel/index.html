分部求和 & 排序不等式

发表于 2026-04-26 更新于 2026-04-28 分类于 数学 ， 数学竞赛
不等式中和式的处理

Abel 变换（分布求和公式）

$$\sum_{i=1}^na_ib_i=\sum_{i=1}^{n-1}S_i(b_k-b_{k+1})+S_nb_n$$

其中 $S_i=\sum_{k=1}^ia_k$。如果记 ΔSk = Sk − Sk − 1，Δbk = bk + 1 − bk，那么上述公式可以写成

$$\sum_{k=1}b_k\Delta S_k=S_nb_n-\sum_{k=1}^nS_k\Delta b_k$$

这与分部积分公式相近，成为分部求和公式。

Abel 不等式

设 a1, …, an, b1, …, bn ∈ ℝ，且 b1 ≥ b2 ≥ … ≥ bn ≥ 0，记 $S_k=\sum_{i=1}^na_i,k=1,2,\dots,n$，且

$$M=\max_{i=1}^n\{S_i\},m=\min_{i=1}^n\{S_i\}$$

证明 Abel 不等式

$$mb_1\leq \sum_{k=1}^na_kb_k\leq Mb_1$$

proof.

$$\sum_{k=1}^na_ib_i=\sum_{k=1}^nS_k(b_k-b_{k-1})\$$

m ≤ Sk ≤ M

Q.E.D.

例题：设 a1, …, an > 0, b1 ≥ b2 ≥ … ≥ bn > 0，且 a1a2…ak ≥ b1b2…bk 对 k = 1, 2, …, n 都成立，证明：

∑ai ≥ sumbi

proof.

$$\sum^k\frac{a_i}{b_i}\geq k\sqrt[k]{\prod^k \frac{a_i}{b_i}}\geq k$$

$$\sum^k (\frac{a_i}{b_i}-1)\geq 0$$

$$\sum b_i(\frac{a_i}{b_i}-1)=\sum_k\sum_{i=1}^k(\frac{a_i}{b_i}-1)(b_k-b_{k+1})\geq 0$$

钟开莱不等式：设 a1 ≥ a2 ≥ … ≥ an > 0 且 $\sum_{i=1}^ka_i\leq \sum_{i=1}^kb_i$，证明

  1. ∑ai2 ≤ ∑bi2

  2. ∑ai3 ≤ ∑aibi2

proof.

设 $A_k=\sum_{i=1}^ka_i$，$B_k=\sum_{i=1}^kb_i$，则

∑ai2 = ∑Ak(ak − ak + 1) ≤ ∑Bk(ak − ak + 1) = ∑aibi

再由柯西不等式，$\sum a_i^2\geq \dfrac{(\sum a_ib_i)^2}{\sum b_i^2}$，得 ∑aibi ≤ ∑bi2

故 ∑ai2 ≤ ∑bi2

对第二问，有

∑ai3 = ∑Ak(ak2 − ak + 12) ≤ ∑Bk(ak2 − ak + 12) = ∑ai2bi

由 Cauchy，$\sum a_i^3\geq \dfrac{(\sum a_i^2b_i)^2}{\sum b_i^2a_i}$，有 ∑ai2bi ≤ ∑bi2ai，故

∑ai3 ≤ ∑bi2ai

例题：n ≥ 2，x1 + … + xn = 0，|x1| + |x2| + … + |xn| = 1，求证

$$|x_1+\frac{x_2}{2}+\frac{x_3}{3}+\dots +\frac{x_n}{n}|\leq \frac{1}{2}-\frac{1}{2n}$$

proof.

我们发现假设左边绝对值里面的东西是正的，那么肯定是负的数越小越往后放左边的值更大，反之取个绝对值也是一样的。这告诉我们什么？

设 Sk = x1 + x2 + … + xk，我们来证

$$|S_k|=|x_1+x_2+\dots+x_k|=|x_{k+1}+\dots+x_n| \\=\frac{1}{2}(|x_1+x_2+\dots+x_k|+|x_{k+1}+\dots+x_n|) \\\leq \frac{1}{2}(|x_1|+|x_2|+\dots+|x_n|)=\frac{1}{2} $$

由 abel 变换，

$$LHS=|\sum_{k=1}^{n-1}S_k(\frac{1}{k}-\frac{1}{k+1})+S_n\cdot \frac{1}{n}|\leq \sum_{k=1}^{n-1}|S_k|(\frac{1}{k}-\frac{1}{k+1})\leq \frac{1}{2}-\frac{1}{2n}$$

例题：ai > 0，i = 1, 2, …, n，$\sum_{i=1}^ka_i\leq k$，求证

$$\sum_{i=1}^n \frac{a_i}{i}\leq 1+\frac{1}{2}+\dots+\frac{1}{n}$$

proof.

$$(a_1-1)+\frac{1}{2}(a_2-1)+\dots+\frac{1}{n}(a_n-1)\leq 0$$

由 abel 变换，

$$LHS\leq \sum_{k=1}^{n-1}(\sum_{i=1}^ka_i-k)(\frac{1}{k}-\frac{1}{k-1})+(\sum_{i=1}^na_i-n)\cdot\frac{1}{n}\leq 0$$

例题：ai > 0，i = 1, 2, …, n，ai + j ≤ ai + aj，求证

$$a_1+\frac{a_2}{2}+\dots+\frac{a_n}{n}\geq a_n$$

proof.

考虑数学归纳法，如果

$$a_1+\frac{a_2}{2}+\dots+\frac{a_{n-1}}{n-1}\geq a_{n-1}$$

那么

$$a_1+\frac{a_2}{2}+\dots+\frac{a_n}{n}\geq a_n$$

就等价于

nan − 1 ≥ (n − 1)an

哎不对怎么放过了。考虑换第二类数学归纳法。

a1 ≥ a1

$$a_1+\frac{a_2}{2}\geq a_2$$

$$a_1+\frac{a_2}{2}+\frac{a_3}{3}\geq a_3$$

…

$$a_1+\frac{a_2}{2}+\frac{a_3}{3}\dotsb+\frac{a_{n-1}}{n-1}\geq a_{n-1}$$

上面的式子相加，有

$$(n-1)a_1+(\frac{n}{2}-1)a_2+(\frac{n}{3}-1)a_3+\dots+(\frac{n}{n-1}-1)a_{n-1}\geq a_1+\dotsb+a_n \\ \iff n(a_1+\frac{a_2}{2}+\frac{a_3}{3}+\dots+\frac{a_{n-1}}{n-1})\geq 2(a_1+\dotsb+a_{n-1}) \\ \iff n(a_1+\frac{a_2}{2}+\frac{a_3}{3}+\dotsb+\frac{a_n}{n})\geq 2(a_1+\dotsb+a_{n-1})+a_n $$

要证明

$$(n-1)(a_1+\frac{a_2}{2}+\dots+\frac{a_n}{n})\geq 2(a_1+\dots+a_n) \\=(a_1+a_{n-1})+(a_2+a_{n-2})+\dotsb+(a_{n-1}+a_1)\geq (n-1)a_n$$

所以

$$a_1+\dots+\frac{1}{n-1}a_{n-1}+\frac{1}{n}a_n\geq \frac{n-1}{n}a_n+\frac{1}{n}a_n=a_n$$

▫

排序不等式

a1 ≤ a2 ≤ … ≤ an

b1 ≤ b2 ≤ … ≤ bn

i1, i2, …, in 是 1, 2, …, n 的一个排列，求证

a1b1 + a2b2 + … + anbn ≥ ai1bi1 + … + ainbin

proof.

a1(b1 − bi1) + … + an(bn − bin) ≥ 0

$$\sum_{k=1}^n(\sum_{j=1}^k(b_j-b{i_j}))(a_k-a_{k+1})\geq 0$$

$$\sum_{k=1}^n(\sum_{j=1}^kb_j-\sum_{j=1}^kb_{i_j})(a_k-a_{k+1})\geq 0$$

Q.E.D
