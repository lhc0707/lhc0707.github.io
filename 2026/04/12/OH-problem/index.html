厉害的羟基数学题

发表于 2026-04-12 更新于 2026-04-26 分类于 数学 ， 数学竞赛

Minkowski 不等式

$$ \sum_{i=1}^n\sqrt{a_i^2+b_i^2}\geq \sqrt{(\sum_{i=1}^na_i)^2+(\sum_{i=1}^nb_i)^2} $$

为勾股定理推论。

并不明显的一次函数

x + y + z = 3，x, y, z ≥ 0，求证

x2 + y2 + z2 + xyz ≥ 4

分析与解：

x2 + y2 + z2 + xyz − 4 = x2 + (y + z)2 − 2yz + xyz − 4 = (x − 2)yz + x2 + (3 − x)2 − 4

对给定的 x，上式是关于 yz 的不超过一次的函数。

$$0\leq yz\leq (\frac{3-x}{2})^2$$

设 f(t) = (x − 2)t + x2 + (3 − x)2 − 4，只要证

$$f(0)\geq 0,f((\frac{3-x}{2})^2)\geq 0$$

$$f(0)=x^2+(3-x)^2-4=2(x-\frac{3}{2})^2+\frac{1}{2}\geq 0$$

$$f((\frac{3-x}{2})^2)=(\frac{x+2}{4})(3-x)^2+(x+2)(x-2)=\frac{1}{4}(x+2)(x-1)^2\geq 0$$

范德蒙德行列式

这样的标题会让这个东西看起来很牛逼，符合标题。

a2(b − c) + b2(c − a) + c2(a − b) = (a − b)(b − c)(a − c)

你真的认识齐次吗？

From PKU 2021

易知 a, b, c > 0，且

$$(a+b-c)(\frac{1}{a}+\frac{1}{b}-\frac{1}{c})=3$$

求 $$(a^4+b^4+c^4)(\frac{1}{a^4}+\frac{1}{b^4}+\frac{1}{c^4})$$ 的最小值。

分析与解：

这啥呀先拆开条件再说说。

$$3=3+\frac{a}{b}+\frac{b}{a}-((\frac{1}{a}+\frac{1}{b})c+(a+b)\frac{1}{c})$$

$$\frac{a}{b}+\frac{b}{a}=(\frac{1}{a}+\frac{1}{b})c+(a+b)\frac{1}{c}$$

然后咋办？你发现这是齐次的！直接设 c = 1，问题就变成了：

条件：

$$(a+b-1)(\frac{1}{a}+\frac{1}{b}-1)=3$$

求：

$$(a^4+b^4+1)(\frac{1}{a^4}+\frac{1}{b^4}+1)$$

对所求直接上 Cauchy，有

$$LHS\geq (\frac{a^2}{b^2}+\frac{b^2}{a^2}+1)^2=(t^2-1)^2$$

t2 − 4t + 4 ≥ 12

$$t\geq 2+\sqrt 3$$

所以

$$\text{原式}\geq 417+240\sqrt 3$$

单射是有用的

from. USTC 2024

某不爱写解析的数学老师的羟基讲义第一题，开幕雷击这一块。

函数 f : ℝ → ℝ，对任意 x, y ∈ ℝ 都有

f(x + f(y)) = f(f(x)) + y

且 f(1) = 2024，求 f(2024)。

分析与解：

函数方程先代数。

f(1 + f(0)) = f(2024)

f(0 + 2024) = f(f(0)) + 1

然后有

f(f(0) + 1) = f(f(0)) + 1 = f(2024)

然后就没有头绪了，但是你发现如果 f(x1) = f(x2)，那么

f(f(x)) + x1 = f(f(x)) + x2

所以 f 是单射，所以 f(0) + 1 = 2024, f(0) = 2023，然后

f(x + 2023) = f(f(x)) = f(2023) + x

于是 f(x) = x + C，带入 f(1) = 2024 得 f(x) = x + 2023，所以

f(2024) = 4047

2026 年了，该咋办呢？

from. USTC 2025

$$(1-x)^{2025}=\sum_{i=0}^{2025}a_ix^i$$

求 $\sum_{i=1}^{2025}\frac{1}{a_i}$

$$a_i=(-1)^i\binom{n}{i}$$

在 n 为奇数的时候，$\sum_{k=0}^n\frac{(-1)^k}{C_n^k}=0$，但是今年是 2026 年了，n 是偶数该咋办？

先写两项，好像有

$$\sum_{k=0}^n\frac{(-1)^k}{C_n^k}=\frac{2n+2}{n+2}=2-\frac{2}{n+2}$$

引理 1：$$\sum_{k=0}^{2n-1}(-1)^{k-1}(k+1)(C_{2n}^k)^{-1}=0$$

证明：$$\frac{k+1}{C_{2n}^k}=\frac{2n+1}{C_{2n+1}^{k+1}}=\frac{2n+1}{C_{2n}^{2n-k}}=\frac{2n-k}{C_{2n}^{2n-k-1}}$$

令 l = 2n − k − 1，那么

$$LHS=\sum_{l=0}^{2n-1}(-1)^{2n-l}\frac{l+1}{C_{2n}^l}=-LHS$$

引理得到证明。

记 $$a_n=\sum_{k=1}^{2n-1}(-1)^{k-1}(C_{2n}^k)^{-1}$$

$$b_n=\sum_{k=1}^{2n-1}(-1)^{k-1}k(C_{2n}^k)^{-1}$$

an + bn = 1

nan − bn = 0

解得 $a_n=\frac{1}{n+1}$，就完事了。
