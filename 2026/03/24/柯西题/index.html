柯西 & 卡尔松 & 赫尔德

发表于 2026-03-24 更新于 2026-04-19 分类于 数学 ， 数学竞赛

不要把答案的一步出结果当成做题的正确姿态。短短一行字的背后可能会浪费掉你两张草稿纸

Prob 1. a, b, c ≥ 0，证明

$$\frac{a^2-bc}{2a^2+b^2+c^2}+\frac{b^2-ac}{a^2+2b^2+c^2}+\frac{c^2-ba}{a^2+b^2+2c^2}\geq 0$$

proof.

猜测在 a = b = c 时取等，验证一下，如果 a = 1, b = c = 0，那么

$$\text{LHS}=\frac{a^2-1}{2a^2+2}+\frac{2(1-a)^2}{3+a^2}=(a-1)(\frac{a+1}{2a^2+2}-\frac{2}{3+a^2})$$

右边那个式子的分子化简之后时 (a − 1)3 ≥ 0，所以等号取不到。

条件等价于

$$\sum_{\text{cyc}}\frac{a^2}{2a^2+b^2+c^2}\geq \sum_{\text{cyc}}\frac{bc}{2a^2+b^2+c^2}$$

放缩的时候要考虑往 a = b = c 上面去配。

发现问题在于分子的符号有正的也有负的，考虑把分子整成一个平方的形式

$$\frac{1}{2}-\frac{a^2-bc}{2a^2+b^2+c^2}$$

的分子就是 (b + c)2，命题就转化为

$$\sum_{\text{cyc}}\frac{(b+c)^2}{2a^2+b^2+c^2}\leq 3$$

由权方和不等式，

$$\frac{(b+c)^2}{2a^2+b^2+c^2}\leq \frac{b^2}{a^2+b^2}+\frac{c^2}{a^2+c^2}$$

就证完了。

这提示我们在分子较为复杂的时候可以通过加减常数的方式让分子和分母做运算，然后利用我们熟知的不等式

Prob 2. $a,b,c\geq 1,\sum \frac{1}{a}=2$，证

$$\sqrt{a+b+c}\geq\sqrt{a-1}+\sqrt{b-1}+\sqrt{c-1}$$

proof.

令 (x, y, z) = (1/a, 1/b, 1/c)，则 x + y + z = 2 且 x, y, z ∈ (0, 1]，要证

$$\sqrt{1/x+1/y+1/z}\geq\sum_{\text{cyc}}\sqrt{\frac{1-x}{x}}$$

其中 1 − x = y + z − 1，1 − x + 1 − y + 1 − z = 1

$$(1-x+1-y+1-z)(\sum_{\text{cyc}}\frac{1}{x})\geq (\sum_{cyc}\sqrt{\frac{1-x}{x}})^2$$

Prob 3. a, b, c > 0，证明

$$\sum\frac{a^3}{a^3+b^3+abc}\geq 1$$

proof.

$$\text{LHS}=\sum_{\text{cyc}}\frac{1}{1+(b/a)^3+(b/a)(c/a)}\geq \frac{9}{3+(b/a)^2+(c/b)^2+(a/c)^2+(b/a)(c/a)+(c/b)(a/b)+(a/c)(b/c)}$$

换元，令 (x, y, z) = (b/a, c/b, a/c)，则 xyz = 1，要证

x3 + y3 + z3 + x/z + y/x + z/y ≤ 6

这显然不一定是对的，说明放过了，但是还原是值得保留的。

$$\sum_{\text{cyc}}\frac{1}{1+x^3+x/z}=\sum_{\text{cyc}}\frac{z}{x+z+x^2/y}=\sum_{\text{cyc}}\frac{yz}{xy+yz+x^2}$$

$$=\sum_{\text{cyc}}\frac{(yz)^2}{yz(xy+yz+zx)}\geq \frac{(xy+yz+zx)^2}{2(x+y+z)+(xy^2+yz^2+zx^2)}$$

就做完了。

Prob 4. a, b, c, d ≥ 0，求证

$$\sum_{\text{cyc}}\frac{a}{b^2+c^2+d^2}\geq \frac{4}{a+b+c+d}$$

proof.

$$(a+b+c+d)\sum_{\text{cyc}}\frac{a}{b^2+c^2+d^2}\geq (\sum_{\text{cyc}}\sqrt{\frac{a^2}{b^2+c^2+d^2}})^2$$

要证

$$\sum_{\text{cyc}}\sqrt{\frac{a^2}{b^2+c^2+d^2}}\geq 2$$

不妨设 a2 + b2 + c2 + d2 = 1，然后你发现取等条件并不是 a = b = c = d，而是 $(a,b,c,d)=(\sqrt 2/2,\sqrt 2/2,0,0)$，这提示我们应当注意放缩的手法。

$$\sqrt{\frac{1-a^2}{a^2}}\leq \frac{1}{2}(\frac{1-a^2}{a^2}+1)$$

取等条件为 a2 = 1/2，倒回来有

$$\sqrt{\frac{a^2}{1-a^2}}\geq 2a^2$$

发现这样的放缩在 a = 0 时也成立，然后就做完了。

对于 ∑cyca = Const 的条件与 ∑cycf(a) ≥ x 要证明的不等式，如果发现取等条件要让一部分为一个值，另一部分为另一个值，那么放缩的目的就是要让 y = f(x) = kx + m 的解为这两个值。此即割线放缩。

Prob. 5. a, b, c, d, e, f > 0，求证

$$\frac{a}{b+c}+\frac{b}{c+d}+\frac{c}{d+e}+\frac{d}{e+a}+\frac{e}{a+f}+\frac{f}{a+b}\geq 3$$

proof.

$$\text{LHS}=\frac{a^2}{ab+ac}+\frac{b^2}{bc+bd}+\frac{c^2}{cd+ce}+\frac{d^2}{de+df}+\frac{e^2}{ef+ea}+\frac{f^2}{fa+fb}$$

$$\geq \frac{(a+b+c+d+e+f)^2}{ab+ac+ae+af+bc+bd+bf+cd+ce+de+df+ef}$$

只要证

(a + b + c + d + e + f)2 − 3(ab + ac + ae + af + bc + bd + bf + cd + ce + de + df + ef) ≥ 0

 ⇔ a2 + b2 + c2 + d2 + e2 + f2 + 2ad + 2be + 2cf − ab − ac − … − ef ≥ 0

其中 2(ab + ac + … + ef) = (a + b + c + d + e + f)2 − ((a + d)2 + (b + e)2 + (c + f)2)，所以只要

$$(a+b+c+d+e+f)^2-\frac{3}{2}(a+b+\dots+f)^2+\frac{3}{2}((a+d)^2+(b+e)^2+(c+f)^2)\geq 0$$

 ⇔ 3((a + d)2 + (b + e)2 + (c + f)2) ≥ (a + b + … + f)2

柯西不等式易证。

Prob. 6. a, b, c > 0，a + b + c = 1，求证

$$\frac{1+a}{1-a}+\frac{1+b}{1-b}+\frac{1+c}{1-c}\leq \frac{2a}{b}+\frac{2b}{c}+\frac{2c}{a}$$

proof.

由 $\frac{1+a}{1-a}=1+\frac{2a}{1+a}$，只要证

$$2a(\frac{1}{b}-\frac{1}{b+c})+2b(\frac{1}{c}-\frac{1}{c+a})+2c(\frac{1}{a}-\frac{1}{a+b})\geq 3$$

即 $$\frac{ac}{b(b+c)}+\frac{ab}{c(c+a)}+\frac{bc}{a(a+b)}\geq \frac{3}{2}$$

同乘分子，即 $$\frac{a^2c^2}{abc(b+c)}+\frac{a^2b^2}{abc(c+a)}+\frac{b^2c^2}{abc(a+b)}\geq \frac{3}{2}$$

$$\text{LHS}\geq \frac{(ab+bc+ca)^2}{abc(b+c+c+a+a+b)}$$

故只要证 (ab + bc + ca)2 ≥ 3abc(a + b + c)

令 (x, y, z) − (ab, bc, ca)，有

(x + y + z)2 ≥ 3(yz + zx + xy)

由 米尔黑德定理，有

a2 + b2 + c2 ≥ ab + bc + ca

a2b + b2c + c2a ≥ 3abc

证毕。

Prob. 7. a, b, c, d > 0，求证

$$\sum_{\text{cyc}}\frac{a}{3a^2+2b^2+c^2}\leq \frac{1}{6}(\sum_{\text{cyc}}\frac{1}{a})$$

proof.

3a2 + 2b2 + c2 = 2(a2 + b2) + (a2 + c2) ≥ 4ab + 2ac

故 $$\text{LHS}\leq \sum_{cyc}\frac{1}{4b+2c}\leq \sum_{cyc}(\frac{m^2}{4b}+\frac{(1-m)^2}{2c})=\sum_{cyc}(\frac{m^2}{4}+\frac{(1-m)^2}{2})\frac{1}{a}$$

要使 $\frac{m^2}{4}+\frac{(1-m)^2}{2}=\frac{1}{6}$，有 $m=\frac{2}{3}$。

Prob. 8. 设 a, b, c > 0，a + b + c = ab + bc + ca，求证

$$\frac{1}{a^2+b+1}+\frac{1}{b^2+c+1}+\frac{1}{c^2+a+1}\leq 1$$

proof.

(a2 + b + 1)(1 + b + c2) ≥ (a + b + c)2

故只要证

3 + (a + b + c) + (a2 + b2 + c2) ≤ (a + b + c)2

3 + a + b + c ≤ 2(ab + bc + ca)

a + b + c ≥ 3

这是因为

(a + b + c)2 ≥ (ab + bc + ca)

Prob. 9. a, b, c > 0，a2 + b2 + c2 = 3，求证

$$\sum\frac{a}{a^2+2a+3}\leq \frac{1}{2}$$

切线放缩易证。

$$\sum_{cyc}\frac{a}{(a^2+1+2(a+1))}\leq \sum_{cyc}\frac{a}{4a+2}$$

只要证

$$\sum_{cyc}\frac{a}{2a+1}\leq 1$$

切线放缩易证

Prob. 10. a, b, c > 0，a2 + b2 + c2 = 3，求证

$$\sum\frac{a}{a^2+2b+3}\leq \frac{1}{2}$$

proof.

$$LHS=\sum_{cyc}\frac{a}{(a^2+1)+2(b+1)}\leq \sum_{cyc}\frac{a}{2(a+b+1)}$$

只要证

$$\sum\frac{b+1}{a+b+1}\geq 2$$

$$\sum\frac{(b+1)^2}{(a+b+1)(b+1)}\geq 2$$

$$LHS\geq \frac{(a+b+c+3)^2}{a^2+b^2+c^2+ab+bc+ca+3(a+b+c)+3}=2$$

Prob. 11. ai ≥ 0，求证

$$\prod_{i=1}^n(1+a_i)\geq (1+(\prod_{i=1}^na_i)^{\frac{1}{n}})^n$$

proof.

发现 n = 2 时即 Cauchy 不等式。

回忆 AM-GM 不等式证明 Cauchy 不等式：

$$\frac{a_i^2}{\sum a^2}+\frac{b_i^2}{\sum b^2}\geq \frac{2|a_ib_i|}{\sqrt{\sum a^2}\sqrt{\sum b^2}}$$

对 n 个式子求和即得。

考虑推广到

$$\frac{a_i^3}{\sum a^3}+\frac{b_i^3}{\sum b^3}+\frac{c_i^3}{\sum c^3}\geq \frac{3a_ib_ic_i}{\sqrt[3]{(\sum a^3)(\sum b^3)(\sum c^3)}}$$

对 i = 1, 2 求和，有

(∑ai3)(∑bi3)(∑ci3) ≥ (a1b1c1 + a2b2c2)3

更一般的，

(a11n + a12n)(a21n + a22n)…(an1n + an2n) ≥ (a11…an1 + a11…an1)

再推广，有

$$\prod_{i=1}^n(a_{i1}^n+\dots+a_{im}^n)\geq (\sum_{j=1}^ma_{1j}a_{2j}\dots a_{nj})^n$$

此即 Calson 不等式。

Prob. 12 即为证明 Calson 不等式。

一种运用均值不等式的做法：

由均值不等式，有

$$(\prod\limits_{j=1}^m\dfrac{x_{j1}}{\sum\limits_{i=1}^nx_{ji}})^{\frac{1}{m}}\leq \frac{1}{m}\sum\limits_{j=1}^m\dfrac{x_{j1}}{\sum\limits_{i=1}^nx_{ji}}$$

同理有

$$(\prod\limits_{j=1}^m\dfrac{x_{j2}}{\sum\limits_{i=1}^nx_{ji}})^{\frac{1}{m}}\leq \frac{1}{m}\sum\limits_{j=1}^m\dfrac{x_{j2}}{\sum\limits_{i=1}^nx_{ji}}$$

…

$$(\prod\limits_{j=1}^m\dfrac{x_{jn}}{\sum\limits_{i=1}^nx_{ji}})^{\frac{1}{m}}\leq \frac{1}{m}\sum\limits_{j=1}^m\dfrac{x_{jn}}{\sum\limits_{i=1}^nx_{ji}}$$

相加，得

$$\sum\limits_{k=1}^n(\prod\limits_{j=1}^m\dfrac{x_{jk}}{\sum_{i=1}^nx_{ji}})^{\frac{1}{m}}\leq 1$$

整理即得 Calson 不等式。

这个不等式再范浩熊的《不等式的秘密》中被称为赫尔德不等式，可以认为是一种误读。

Prob. 13.

求 $$\frac{xyz}{(1+4x)(9x+y)(4y+z)(9z+1)}$$ 的最大值。

sol.

$$(1+4x)(9x+y)(4y+z)(9z+1)\geq (\sqrt[4]{3^4\times 2^2xyz}+\sqrt[4]{2^2xyz})^4=1024xyz$$

故 $$\frac{xyz}{(1+4x)(9x+y)(4y+z)(9z+1)}\leq \frac{1}{1024}$$

等号条件存在。

Prob. 14. a, b, c > 0，求证

$$\frac{a}{\sqrt{a^2+8bc}}+\frac{b}{\sqrt{b^2+8ca}}+\frac{c}{\sqrt{c^2+8ab}}\geq 1$$

proof.

LHS2(∑cyca(a2 + 8bc)) ≥ (a + b + c)3

故只要证 $$\frac{(a+b+c)^3}{\sum_{cyc}a(a^2+8bc)}\geq 1$$

即 (a + b + c)3 ≥ a3 + b3 + c3 + 24abc

易证。

Prob. 15. 证明权方和不等式

$$\sum\frac{a_i^{m+1}}{b_i^m}\geq \frac{(\sum a_i)^{m+1}}{(\sum b_i)^m}$$

proof.

$$LHS\cdot (\sum b_i)^m\geq (\sum \sqrt[m+1]{\frac{a_i^{m+1}}{b_i^m}b_i^m})^{m+1}=(\sum a_i)^m+1$$

Prob. 16 废物题。

Prob. 17 证明幂平均不等式。

定义

$$M_p=(\frac{1}{n}\sum_{i=1}^na_i^p)^{\frac{1}{p}}$$

要证对 p > q，有 Mp ≥ Mq。

proof.

对于 p > q > 0，两边同时做 p 次方，有

$$\frac{1}{n}\sum_{i=1}^na_i^p\geq (\frac{1}{n}\sum_{i=1}^na_i^q)^{p/q}$$

$$\frac{1}{n}\sum_{i=1}^n(a_i^q)^{p/q}\geq (\frac{1}{n}\sum_{i=1}^na_i^q)^{p/q}$$

记 xi = aiq, t = p/q > 1，只要证

$$\frac{1}{n}\sum_{i=1}^nx_i^t\geq (\frac{1}{n}\sum_{i=1}^nx_i)^t$$

对 f(x) = xt 求二阶导，

f″(x) = t(t − 1)xt − 2 > 0

琴声不等式易证。对 0 > p > q 的情况同理。对 p > 0 > q 的情况易证。

这个还可以用 Calson 不等式证明。

两边同时做 pq 次幂，有

$$(\frac{1}{n}\sum_{i=1}^na_i^p)^q\geq (\frac{1}{n}\sum_{i=1}^na_i^q)^p$$

$$n^{p-q}(\sum_{i=1}^na_i^p)^q\geq (\sum_{i=1}^na_i^q)^p$$

$$LHS=(\sum_{i=1}^n1)^{p-q}(\sum_{i=1}^na_i^p)^q\geq (\sum_{i=1}^n\sqrt[p]{1^{p-q}(a_i)^pq})^p=RHS$$

Prob. 18. a + b + c = 1，求 $\sum_{cyc}(a+\frac{1}{a})^3$ 的最小值。

$$\text{原式}\geq \frac{1}{9}(a+b+c+1/a+1/b+1/c)^3\geq \frac{1}{9}(1+9)^3=\frac{1000}{9}$$

Prob. 19. 求证赫尔德不等式

对于 p, q ∈ ℚ，p, q > 1，$\frac{1}{p}+\frac{1}{q}=1$，{an}, {bn} 为正实数序列，有

$$(\sum_{i=1}^ka_i^p)^\frac{1}{p}(\sum_{i=1}^kb_i^q)^{\frac{1}{q}}\geq \sum_{i=1}^ka_ib_i$$

proof.

发现在 p = q = 2 时即为柯西不等式。

设 $p=\frac{m}{n}$，则 $q=\frac{m}{m-n}$。

$$(\sum_{i=1}^ka_i^p)^\frac{n}{m}(\sum_{i=1}^kb_i^q)^\frac{m-n}{m}\geq \sum_{i=1}^na_ib_i$$

$$(\sum_{i=1}^ka_i^p)^n(\sum_{i=1}^kb_i^q)^{m-n}\geq (\sum_{i=1}^ka_ib_i)^m$$

此即 Calson 不等式。
